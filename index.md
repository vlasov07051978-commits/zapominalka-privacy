# Privacy Policy
Запоминалка  
Последнее обновление: 25 февраля 2026

## 1. Общая информация

Приложение «Запоминалка» является полностью офлайн-приложением для создания и хранения заметок.

Мы уважаем приватность пользователей и не собираем, не передаём и не обрабатываем персональные данные.

## 2. Какие данные мы собираем

Приложение не собирает:

- персональные данные
- номера телефонов
- email
- геолокацию
- контакты
- фотографии
- данные устройства для аналитики
- рекламные идентификаторы

Все заметки создаются и хранятся исключительно на устройстве пользователя.

## 3. Передача данных третьим лицам

Приложение:

- не использует серверы
- не передаёт данные в облако
- не подключается к сторонним сервисам
- не содержит рекламы
- не использует аналитические SDK

## 4. Покупка PRO-версии

При покупке PRO-версии используется платёжная инфраструктура RuStore.

Приложение получает только подтверждение факта покупки.  
Платёжные данные обрабатываются RuStore в соответствии с их политикой конфиденциальности.

## 5. Хранение данных

Все пользовательские заметки хранятся локально на устройстве.  
Удаление приложения приводит к удалению данных.

## 6. Изменения политики

В случае обновления политики конфиденциальности новая версия будет опубликована на этой странице.
